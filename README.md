# SQMS – Backend Service

## Purpose
This branch contains the backend API and database implementation.

## Responsibilities
- Authentication (JWT)
- User management (Admin, Staff, Customer)
- Queue logic & ticketing
- Branch management
- Notifications integration
- Appointment scheduling logic
- Database design & schema

## Tech Stack
- Node.js (Express) / Django
- PostgreSQL / MongoDB
- JWT Authentication

## Project Structure
backend/
├── controllers/
├── models/
├── routes/
├── services/
├── config/
└── app.js / manage.py

## API Documentation
API endpoints will be documented using Swagger or Postman.
