# Smart Queue Management System (SQMS)

Smart Queue Management System (SQMS) is a web application that enables businesses 
(banks, hospitals, service centers) to manage queues efficiently.

## Purpose of This Branch
This branch contains the final, stable, production-ready version of the system.

Only tested and approved code is merged into `main`.

## Core Features
- User Roles (Admin, Staff, Customer)
- Virtual Queue & Digital Ticketing
- Real-time Queue Status
- Multi-Branch Support
- Appointment Scheduling (Optional)
- Analytics & Reports

## Tech Stack
- Frontend: React
- Backend: Node.js (Express) / Django
- Database: PostgreSQL / MongoDB
- Authentication: JWT
- Notifications: Twilio / Firebase
- Hosting: AWS / Vercel / Firebase
